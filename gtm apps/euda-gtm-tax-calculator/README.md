# Generation Rent

This workspace is for developing the "Generation Rent" project.

## Project Guidelines

Everything built in this workspace should follow these rules:

1.  **UI Components**: Use [Shadcn](https://ui.shadcn.com/) for all app and tool development.
2.  **Internationalization**: Use [Wuchale](https://wuchale.dev/) for i18n support.
3.  **Branding**: Adhere to the Spanish-inspired color palette (Terracotta, Olive, Warm Cream) defined in `branding.css` and `.agent/instructions.md`.

## Setup
Refer to `.agent/instructions.md` for the full brand configuration and Tailwind CSS 4.0 theme variables.
